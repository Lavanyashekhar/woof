🐾 About the App

Woof is a modern Android application built using Jetpack Compose and Material Design 3 (Material You) principles. The app displays a scrollable list of dog profiles where users can expand cards to see more details and mark dogs as favourites with smooth animations.

The project focuses on integrating theming, state management, and animations to create a polished and responsive UI experience.

✨ Key Features

🎨 Material 3 Light & Dark Theme Support

🐶 Expandable Dog Cards

❤️ Animated Favourite Toggle

🔄 Smooth Expand/Collapse Animation

🎯 LazyColumn Optimized List

📱 Responsive UI Design

🧩 Clean State Management

🎬 Animation Highlights

animateContentSize() for smooth card expansion

animateFloatAsState() for arrow rotation

animateColorAsState() for favourite icon transition

All animations are designed to communicate state changes clearly and enhance user interaction.

🎨 Theming Implementation

The app uses:

MaterialTheme.colorScheme

MaterialTheme.typography

MaterialTheme.shapes

Light and Dark color schemes

Surface and SurfaceVariant tokens for visual hierarchy

No hardcoded colors are used — ensuring brand consistency and scalability.
